COME INSTALLARE Ublog - MySQL version

Requisiti:

- server web con supporto ASP e database MySQL server

L'installazione di Ublog è molto semplice e comprende pochi passaggi:

1)

- decomprimere tutti i file lasciando intatta la struttura della directory.
- aprire il file "common.asp", contenuto nella cartella "include", ed editare le righe 25-30:

MySQLSVR = "xxx.xxx.xxx.xxx" ( indirizzo IP oppure nome dell'host su cui risiede MySQL server )
MySQLPRT = 3306 ( porta di MySQL - default 3306 )
MySQLUID = "username" ( nome utente MySQL )
MySQLPWD = "password" ( password MySQL )
MySQLDB = "UblogMySQL" ( nome del database MySQL )
MySQLOPT = 16386 ( MyODBC valore opzionale )

inserendo una connessione al db valida.
- sempre nel file "common.asp" editare le righe 45-51 inerenti alla localizzazione della cartella per l'upload delle immagini. Tale cartella deve avere i permessi di scrittura abilitati.
- copiare tutti i file e le cartelle "include" ed "immagini" sullo spazio web.
- uploadare il file "UblogMySQL.sql", contenuto nella cartella "database", su MySQL server.

ATTENZIONE il file in questione include esclusivamente la creazione delle tabelle necessarie al db ma non la creazione ex-novo del db!

2)

- raggiungere tramite browser l'indirizzo http://www.tuosito.com/configura.asp ed accedere alla sezione amministrativa di Ublog con i seguenti username e password:

  • username : admin
  • password : admin
- configurare il weblog.


Note:

- si consiglia di modificare subito username e password di accesso alla sezione amministrativa.
- se si abilita la notifica via email di avvenuto inserimento di nuovo blog o nuovo commento è necessario aver installato un componente apposito (CDonts).

3)

- per la configurazione dell'aspetto grafico è sufficiente editare il file styles.css inserito nella cartella "include" e modificare colori e tipologia di font.
- i file header.asp, footer.asp e metatag.in, inseriti nella cartella "include", consentono una migliore integrazione dell'applicazione nel proprio sito.
