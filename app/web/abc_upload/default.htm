ABCUpload Examples

The following examples demonstrate how to use ABCUpload. They take you through from the most basic of uploads to a full featured upload with progress bar and insertion of BLOB data into a database. The structure of the HTML is designed to be as simple as possible to avoid distracting from the basic functionality. All the examples are explained in detail in the ABCUpload documentation.

  • Basic Upload - shows how a basic upload works.
  • Two Image Upload - shows how to upload two image files.
  • Mixed Field Upload - shows how to upload a form containing a mix of standard form and file input fields.
  • Upload to File Server -shows how to upload a file to a file server preserving any Macintosh or Unicode attributes of the original file.
  • Pure HTML Progress Bar Upload - shows how to show a Pure HTML Progress Bar during an upload.
  • Upload to Database - shows how to upload an image and insert it into a database.
  • Single Page Upload - shows how to combine the form and form processing pages.

If you are using ImageGlue you may be interested in the IGExamples.zip file. This contains the ImageGlue example web site adapted for use with ABCUpload.
